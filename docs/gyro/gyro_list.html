AMP_PHI_N

Definition

Initial amplitude of finite-\(n\) modes.

Comments

  • DEFAULT = 1e-4

  • For linear simulations, the value is unimportant

  • For nonlinear runs, this will sometimes need to be reduced to a smaller value.
