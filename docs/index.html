The General Atomics GACODE SuiteÔÉÅ

GACODE refers to a collection of serial and HPC software developed by the Theory Group at General Atomics, and distributed through the GACODE source code repository at GitHub. There are numerous public installations of GACODE (via AToM or local maintainers). You can also install and build the GACODE suite from sources to run on your desktop or laptop.
