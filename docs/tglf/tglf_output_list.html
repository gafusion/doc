Alphabetical list for out.tglf files

density_spectrum

Description

Gyro-bohm normalized density fluctuation amplitude spectra, \({\frac {a \delta \tilde{n}_{s,k_y}}{\rho_s n_{e0}}}\).

Comments

  • columns: density(ns=1,..,ns_in)

eigenvalue_spectrum

Description

Gyro-bohm normalized eigenvalue spectra

Comments

  • columns: gamma(nm=1), freq(nm=1), gamma(nm=2), freq(nm=2)

field_spectrum

Description

Gyro-bohm normalized field fluctuation intensity spectra per mode

Comments

  • index limits: nky, nmodes

intensity_spectrum

Description

Gyro-bohm normalized intensity fluctuation amplitude spectra per mode

Comments

  • columns: density, temperature, parallel velocity, parallel energy

  • index limits: ns, nky, nmodes

nsts_crossphase_spectrum

Description

Density-temperature cross phase spectra per mode for each species

Comments

  • index limits: nky, nmodes

QL_flux_spectrum

Description

QL weights per mode, species, field and type (type = particle, energy, toroidal stress, parallel stress, exchange)

Comments

  • index limits: nky, nmodes, ns, field, type

sat_geo_spectrum

Description

Saturation model geometry factor per mode \({\frac {<\phi|\phi>}{<\phi| B^2/B_{unit}^2 |\phi>}}\)

Comments

  • index limits: nky, nmodes

scalar_saturation_parameters

Description

This output file has all of the scalar saturation parameters used for different settings of SAT_RULE, UNITS, ALPHA_ZF

spectral_shift

Description

kx spectral shift model is used when ALPHA_QUENCH=0 and ALPHA_E=1.0. Note: The mode for the spectral shift, \(kx_e={\frac{<\phi| k_x/k_y |\phi>}{<\phi|\phi>}}\), depends on which saturation model is being used: SAT_RULE and UNITS settings

Comments

  • index limits: nky

sum_flux_spectrum

Description

flux spectrum summed over nmodes. A simple sum over nky axis will give total flux.

Comments

  • index limits: nky

  • columns: particle flux, energy flux, toroidal stress, parallel stress, exchange

temperature_spectrum

Description

Gyro-bohm normalized temperature fluctuation amplitude spectra

Comments

  • index: nky, ns

  • columns: temperature(ns=1,..,ns_in)
